# Calculadora 

##  Descripción

Este proyecto es una calculadora básica hecha con HTML, CSS y JavaScript.  
Permite realizar operaciones matemáticas simples entre dos números.



##  Tecnologías usadas

- HTML = estructura  
- CSS = diseño  
- JavaScript = funcionamiento  



##  Funcionalidades

- Ingresar números usando botones  
- Seleccionar una operación:
  - Suma (+)
  - Resta (-)
  - Multiplicación (*)
  - División (/)
- Mostrar el resultado en pantalla  
- Botón para limpiar la calculadora  



##  Lógica del programa

El programa funciona de esta manera:

1. Guarda el primer número  
2. Guarda el operador  
3. Guarda el segundo número  
4. Realiza el cálculo  
5. Muestra el resultado  



##  Estructura del proyecto

/proyecto-calculadora
 -index.html
 -style.css
 -script.js

 


##  Cómo ejecutar

1. Descarga o clona el proyecto  
2. Abre el archivo `index.html`  
3. Usa la calculadora en el navegador
